API — Listas Hospitalarias Venezuela 2026

Acceso programático a los datos de pacientes en hospitales y refugios

Buscar Cruces con desaparecidos API

Los datos están disponibles como archivos JSON estáticos, sin necesidad de autenticación ni clave API. Se actualizan cada vez que se agregan nuevas listas al sistema.

Base URL: https://scimoncfd.github.io/terremoto-venezuela-ocr

Archivos disponibles

GET /data.json Todos los registros hospitalarios

Campos

nNombre completo
hHospital o refugio
ciCédula de identidad
eEdad
sSexo (M/F)
dDiagnóstico
fFecha de la lista
uURL de la lista original (imagen o PDF)

Ejemplo

curl "https://scimoncfd.github.io/terremoto-venezuela-ocr/data.json"
Descargar data.json
GET /api/cruces.json Personas en hospitales también reportadas como desaparecidas

Campos

nombreNombre completo
hospitalHospital o refugio donde aparece
cedulaCédula de identidad
estadosin-contacto, localizado o en-hospital
fecha_listaFecha de la lista hospitalaria
fecha_reporteFecha en que fue reportado como desaparecido
fuente_urlURL de la lista hospitalaria original

Ejemplo

curl "https://scimoncfd.github.io/terremoto-venezuela-ocr/api/cruces.json" # Solo los que aún no tienen contacto (con jq) curl "https://scimoncfd.github.io/terremoto-venezuela-ocr/api/cruces.json" | jq '[.[] | select(.estado == "sin-contacto")]'
Descargar cruces.json
GET /admin/api/cruces-con-telefono Cruces con número de teléfono — solo admins

Autenticación

X-Admin-KeyHeader requerido — clave de administrador

Parámetros

pendientes1 (default) — solo los que aún no han sido localizados · 0 — todos los que tienen teléfono

Campos devueltos

nombreNombre completo
hospitalHospital o refugio donde aparece
fuente_urlURL de la lista hospitalaria original
fecha_listaFecha de la lista
telefonoNúmero de contacto del familiar
nombre_contactoNombre del familiar que reportó la desaparición
estadosin-contacto o localizado

Ejemplos

# Solo los pendientes con teléfono curl -H "X-Admin-Key: CLAVE" "https://listashospitalarias.pythonanywhere.com/admin/api/cruces-con-telefono" # Todos los que tienen teléfono curl -H "X-Admin-Key: CLAVE" "https://listashospitalarias.pythonanywhere.com/admin/api/cruces-con-telefono?pendientes=0"
