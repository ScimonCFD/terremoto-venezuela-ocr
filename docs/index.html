Esta herramienta digitaliza las listas de pacientes usando reconocimiento de texto automático. Esto significa que:

  • Puede haber errores de lectura en nombres, edades o diagnósticos.
  • No están incluidas las listas de todos los hospitales del país.
  • La información puede haber cambiado desde que se publicó la lista original.
  • La ausencia de un nombre no confirma que la persona no esté hospitalizada.

Si encuentras un resultado, usa "Ver lista original" para confirmarlo con tus propios ojos. Ante cualquier duda, comunícate directamente con el hospital.
